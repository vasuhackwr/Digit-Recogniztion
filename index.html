About the Application

Digit Recognition is a process that converts scanned documents images into electronically understandable form. Thus, enabling computers to recognize text present in the images. This is a Deep Learning Project based on Convolutional Neural Networks Sequential model.

The Handwritten MNIST dataset (Modified National Institute of Standards and Technology) has been loaded using tensorflow keras API for training and testing of the model. This is a dataset of 60,000 28x28 grayscale images of the 10 digits, along with a test set of 10,000 images.

It is a 8 layer model with Conv2d, MaxPool2d, Droupout, Flatten and Dense layer combination. Input layer has 32 neurons and the output layer has 10 neurons with 10 different classes. 15 epochs are used. 'sparse_categorical_ crossentropy' is loss function and 'adam' is used for optimization. Model gives an accurancy of 99.21% on testing dataset.
